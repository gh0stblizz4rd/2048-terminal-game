# 2048 Game

A command line version of the [2048 game](https://play2048.co/) made using python.

### Run the game

    git clone repolink
    cd reponame
    pip install colored getkey
    python3 main.py

### Dependencies

- [colored](https://pypi.org/project/colored/)
- [getkey](https://pypi.org/project/getkey/)
